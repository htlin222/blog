### 建立一個R project
![image_22-08-31_20_09_59](https://i.imgur.com/5IehENz.png)
* 選擇Book project using bookdown
* 在下一步select HTML book format 選擇 `bs4_book`
* 在`_bookdown.yml`這個文件裡加入：
```
output_dir:"docs"
```
這樣我們輸出的檔案就會在docs這個資料夾裡，才能被gh-page找到
* 編輯你要的markdown文件
* 在左側的panel選擇build
